# יומן דחסידי ויזניץ

## תיאור הפרויקט

אפליקציית ווב לניהול והצגת יומני Google Calendar עבור קהילת חסידי ויזניץ. האפליקציה מאפשרת צפייה ביומנים מרובים, הוספת אירועים חדשים והתאמה אישית של התצוגה.

## תכונות עיקריות

### 📅 ניהול יומנים מרובים
- צפייה ביומנים מרובים של Google Calendar במקום אחד
- בחירה גמישה של איזה יומנים להציג
- יומנים זמינים:
  - פרשיות וימי החודש בעברית
  - שמחות ומעמדי בית ויזניץ
  - מעמדי קהילות הקודש
  - חתונות
  - בר מצוות

### 🎨 עיצוב מתקדם
- ממשק משתמש מודרני ויפה בעברית
- תמיכה במצב מסך מלא
- עיצוב רספונסיבי לכל הגדלי המסך
- צבעי נושא מותאמים לזהות הקהילה
- אנימציות חלקות ואלגנטיות

### ➕ הוספת אירועים
- מודאל להוספת אירועים חדשים
- טופס Google Forms מובנה
- מערכת אישור ידני לאירועים חדשים
- הודעה ברורה למשתמשים על תהליך האישור

### 🔧 תכונות נוספות
- עדכון תאריך אוטומטי
- סייד-בר מתקפל
- סטטיסטיקות של יומנים מוצגים
- מצב טעינה אלגנטי

## טכנולוגיות

- **HTML5** - מבנה העמוד
- **CSS3** - עיצוב מתקדם עם CSS Variables, Flexbox ו-Grid
- **JavaScript ES6** - פונקציונליות דינמית
- **Google Calendar API** - הטמעת יומני Google
- **Google Forms** - מערכת הוספת אירועים
- **Assistant Font** - פונט עברי איכותי מ-Google Fonts

## התקנה והפעלה

1. **הורדת הפרויקט**:
   ```bash
   git clone <repository-url>
   cd calander-viznitz
   ```

2. **פתיחת האתר**:
   - פתח את `index.html` בדפדפן
   - או השתמש בשרת מקומי:
   ```bash
   # עם Python
   python -m http.server 8000
   
   # עם Node.js
   npx serve .
   ```

3. **צפייה באתר**:
   - פתח את הדפדפן וגש ל-`http://localhost:8000`
   - או פתח ישירות את `index.html`

## שימוש באפליקציה

### צפייה ביומנים
1. בחר יומנים מהרשימה בסייד-בר
2. השתמש בכפתור הסויצ'ים כדי להפעיל/לכבות יומנים
3. השתמש בכפתור "מסך מלא" להסתרת הסייד-בר

### הוספת אירוע חדש
1. לחץ על כפתור "הוסף אירוע"
2. מלא את הטופס המובנה
3. האירוע יישלח לבדיקה ואישור
4. לאחר אישור, האירוע יופיע ביומן

## קונפיגורציה

### הוספת יומנים חדשים
ערוך את המערך `calendars` בקובץ `script.js`:

```javascript
const calendars = [
    {
        id: 'calendar-id@group.calendar.google.com',
        name: 'שם היומן',
        color: '#צבע-בהקס',
        priority: true/false
    }
];
```

### שינוי צבעי נושא
ערוך את המשתנים ב-CSS בקובץ `styles.css`:

```css
:root {
    --primary: #8b2635;
    --secondary: #a0306e;
    /* ... */
}
```
